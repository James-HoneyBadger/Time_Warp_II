# Keyboard Shortcuts — Time Warp II

Printable reference for all keyboard shortcuts in the Time Warp II IDE.

---

## Program Execution

| Shortcut | Action |
|----------|--------|
| **F5** | Run program |
| **Escape** | Stop program |

## File Operations

| Shortcut | Action |
|----------|--------|
| **Ctrl+N** | New file |
| **Ctrl+O** | Open file |
| **Ctrl+S** | Save |
| **Ctrl+Shift+S** | Save As |
| **Ctrl+Q** | Exit |

## Editing

| Shortcut | Action |
|----------|--------|
| **Ctrl+Z** | Undo |
| **Ctrl+Y** | Redo |
| **Ctrl+Shift+Z** | Redo (alternative) |
| **Ctrl+X** | Cut |
| **Ctrl+C** | Copy |
| **Ctrl+V** | Paste |
| **Ctrl+A** | Select All |
| **Tab** | Auto-complete keyword |
| **Enter** | Smart auto-indent |

## Search & Navigation

| Shortcut | Action |
|----------|--------|
| **Ctrl+F** | Find |
| **Ctrl+H** | Replace |
| **Ctrl+G** | Go to Line |
| **Ctrl+Shift+P** | Command Palette |

## Zoom

| Shortcut | Action |
|----------|--------|
| **Ctrl+Scroll Up** | Increase font size |
| **Ctrl+Scroll Down** | Decrease font size |
| **Ctrl+Plus** | Increase font size |
| **Ctrl+Minus** | Decrease font size |
| **Ctrl+=** | Increase font size (alternative) |

## Context Menu

| Action | Access |
|--------|--------|
| Right-click in editor | Cut, Copy, Paste, Select All, Go to Line, Find, Replace |

## Command Palette (Ctrl+Shift+P)

The command palette provides searchable access to all actions. Type to filter:

- New File, Open File, Save File, Save File As
- Run Program, Stop Program
- Find, Replace, Go to Line
- Undo, Redo, Select All
- Clear Editor, Clear Output, Clear Graphics
- Fold All Blocks, Unfold All
- Toggle Split Editor
- Export Canvas as PNG, Export Canvas as SVG
- Run Smoke Test, Show Error History
- Quick Reference, Keyboard Shortcuts
- About Time Warp II, Exit
- Theme: Light, Dark, Classic, Solarized Dark, Solarized Light, Monokai, Dracula, Nord, High Contrast
- Font Size: Tiny, Small, Medium, Large, Extra Large, Huge, Giant
